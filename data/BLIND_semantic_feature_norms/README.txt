This archive contains the semantic norms collected from comparable
congenitally blind and sighted Italian subjects that are described in
the following article:
 
Alessandro Lenci, Marco Baroni, Giulia Cazzolli and Giovanna
Marotta. 2013. BLIND: A set of semantic feature norms from the
congenital blind. Behavior Research methods. DOI
10.3758/s13428-013-0323-4.

The appendices from this article, providing details about the data in
the archive, are included.

The archive contains the following files:

README.txt
appendices.pdf
concept-feature-measures.txt
concept-measures.txt
concepts-cosine-matrix.txt
feature-measures.txt
production-data.txt

Refer to Appendix D of the article for a detailed description of the
other files in the archive.
